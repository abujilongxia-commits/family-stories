## 🚀 快速開始

### 1. 預覽網站

在瀏覽器中打開 `index.html`：

```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

### 2. 部署到 GitHub Pages

```bash
# 初始化 Git 倉庫
git init
git add .
git commit -m "Initial commit: 家庭小確幸網站"

# 創建 GitHub 倉庫並推送
git remote add origin https://github.com/abujilongxia-commits/family-stories.git
git branch -M main
git push -u origin main

# 啟用 GitHub Pages
# 前往 GitHub 倉庫 → Settings → Pages → 選擇 main branch
```

### 3. 訪問網站

部署完成後，網站將在以下網址可用：

```
https://abujilongxia-commits.github.io/family-stories/
```
