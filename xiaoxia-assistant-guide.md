# 🦐 蝦蝦助手 - 家庭照片自動處理指南

## 🎯 任務說明

當恩齊傳送家庭照片到 Telegram 時，蝦蝦助手需要：

1. 📸 **接收照片** - 自動保存
2. 🔍 **分析內容** - 識別人、事、時、地、物
3. 📝 **編寫故事** - 50-100 字溫暖故事
4. 🏷️ **分類主題** - 匹配 15 大主題
5. 🎨 **生成 AI** - 宮崎駿風格對比圖
6. 🌐 **更新網站** - 自動添加到畫廊

---

## 📋 處理流程

### 步驟 1：接收照片
```
當收到 Telegram 照片時：
1. 保存原始文件
2. 記錄接收時間
3. 分配照片 ID
```

### 步驟 2：分析照片內容
```
分析要素：
- 👤 人物：爸爸、媽媽、哥哥、妹妹、爺爺、奶奶
- 📍 場景：客廳、餐廳、景點、市場
- 🎯 活動：用餐、遊戲、出遊、購物
- 💝 情感：歡樂、溫馨、感動、幸福
- 🏷️ 物品：冰淇淋、紅包、撲克牌、保齡球
```

### 步驟 3：編寫溫暖故事
```
故事結構（50-100 字）：
1. 時間 + 地點
2. 人物 + 活動
3. 溫暖細節
4. 情感表達

範例：
「週末的午後，全家圍坐在客廳的紅色木沙發上，
分享著杜老爺冰淇淋。孩子們開心地比著 YA 手勢，
茶幾上擺滿了零食，溫暖的歡笑聲充滿了整個客廳。
這就是我們最珍貴的小確幸。」
```

### 步驟 4：分類主題
```
匹配 15 大主題：

| 關鍵詞 | 主題 |
|--------|------|
| 客廳、沙發、聚餐 | 家庭聚會 |
| 夫妻、用餐 | 夫妻情誼 |
| 電影、景點 | 親子出遊 |
| 紅包、春聯 | 春節傳統 |
| 撲克牌、遊戲 | 家庭遊戲 |
| 四代同堂、合照 | 家族團聚 |
| 市場、購物 | 逛街購物 |
| 燈籠、市場 | 節慶市場 |
| 甜品、餐廳 | 美食甜品 |
| 高爾夫、草坪 | 休閒運動 |
| 保齡球、棒球 | 兒童運動 |
| Moomin、景點 | 景點打卡 |
| 花園、鬱金香 | 花園漫步 |
| 學校、日常 | 成長記錄 |
| 爺爺奶奶、多代 | 多代同堂 |
```

### 步驟 5：生成 AI 對比圖
```
提示詞模板：

宮崎駿動畫風格，台灣家庭 [場景]，
[人物描述]，[活動描述]，[情感氛圍]，
手繪質感，柔和 [色調]，自然光線，
細節豐富，台灣家庭日常，吉卜力工作室風格，
龍貓風格溫馨，療癒系，家庭愛

範例：
宮崎駿動畫風格，台灣家庭客廳聚會，
全家圍坐沙發享用冰淇淋，孩子們開心比 YA，
溫暖歡樂氛圍，手繪質感，柔和奶油黃色調，
自然光線，細節豐富，台灣家庭日常，
吉卜力工作室風格，龍貓風格溫馨，療癒系，家庭愛
```

### 步驟 6：更新網站
```
更新內容：
1. 添加到 stories/stories.json
2. 更新 gallery/index.html
3. 生成預覽縮圖
4. 更新進度追蹤
```

### 夫妻情誼
```
[時間]，爸爸和媽媽[活動]在[地點]。
兩人[互動細節]，[溫暖對話或動作]。
[情感表達]，恩愛日常。
```

### 親子出遊
```
[時間]，帶著孩子們來到[地點]。
[活動內容]，孩子們[反應]。
[溫暖時刻]，珍貴的親子回憶。
```

### 春節傳統
```
春節期間，[活動]在[地點]。
[傳統元素：紅包、春聯等]，
[家庭成員互動]，[文化傳承意義]。
```

## 🎨 AI 圖像生成指令

### 使用 NK Images
```bash
curl "https://nkimages.com/api/public/images?source=clawhub&q=[提示詞]&per_page=1" \
  -H "Authorization: Bearer [KEY]"
```

### 使用 EachLabs
```bash
curl -X POST "https://api.eachlabs.ai/v1/images/generations" \
  -H "Authorization: Bearer [KEY]" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "[提示詞]",
    "style": "miyazaki",
    "n": 1
  }'
```
