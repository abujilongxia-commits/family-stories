## 🎨 AI 圖像生成流程

### 步驟 1：分析故事

**讀取故事內容，提取關鍵元素：**
- 場景（客廳、餐廳、花園...）
- 人物（爸爸、媽媽、孩子...）
- 動作（吃冰淇淋、擁抱、拍照...）
- 情感（溫暖、快樂、浪漫...）

### 步驟 2：創建提示詞

**根據故事內容，創建 5-6 個變體提示詞：**

**範例（photo1 家庭冰淇淋聚會）：**

```
提示詞 1：
宮崎駿動畫風格，台灣家庭客廳聚會，紅色木製沙發，
多個家庭成員拿著冰淇淋甜筒，快樂笑容，溫暖手繪質感，
柔和色彩，自然光線從窗戶灑入

提示詞 2：
宮崎駿動畫風格，孩子特寫，鼻尖沾著冰淇淋，
咯咯笑聲，大眼睛閃爍光芒，手繪質感，細節豐富

提示詞 3：
宮崎駿動畫風格，爸爸坐在沙發上，手拿冰淇淋，
看著孩子們，溫暖笑容，客廳背景，家庭愛

提示詞 4：
宮崎駿動畫風格，媽媽從廚房端出更多甜點，
走向客廳，溫柔表情，家庭聚會場景

提示詞 5：
宮崎駿動畫風格，全家福，所有人舉起冰淇淋甜筒，
乾杯姿勢，幸福笑容，溫暖療癒，吉卜力風格
```

### 步驟 3：批量生成

**使用 NK Images 批量生成：**
```bash
# 每個場景生成 28 張（5 個場景 = 140 張）
./generate-miyazaki-batch.sh
```

### 步驟 4：篩選與組織

**從生成的圖像中：**
1. 篩選最符合故事的 5-6 張
2. 保存到對應文件夾
3. 更新映射配置
